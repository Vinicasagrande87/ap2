Estrutura do Sistema

O projeto foi estruturado para garantir uma navegação fluida e modularizada:

Home: Página inicial com foco em conversão e destaque dos produtos.

Usuários: Módulo de autenticação e gerenciamento de perfil.

Produtos: Detalhamento individual com foco na experiência visual e gestão de estoque.

Entrega: Fluxo otimizado para coleta de informações logísticas.

Pagamento: Módulo seguro que suporta diversas modalidades de liquidação financeira.
